# amazon-demo

A full-stack e-commerce web app where you can order your favorite product from the shop. You have options for lots of items. Pay online using your credit card and get your product safely from home.

# Technology: 

* JavaScript, React, Node.js, Express.js, Bootstrap, MongoDB, Firebase Authentication, Stripe.js, Heroku.
